# Home Assistant Custom Cards

A collection of custom dashboard cards for Home Assistant.

## Cards

| Card | Description |
|------|-------------|
| [temperature-dashboard](./temperature-dashboard/) | Multi-room temperature graph with furnace heating/cooling activity overlay |

## Requirements

- [ApexCharts Card](https://github.com/RomRider/apexcharts-card) - Install via HACS

## Usage

Each card folder contains:
- `card.yaml` - The card configuration (copy to your dashboard)
- `README.md` - Card-specific documentation and entity requirements
